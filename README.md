## UCCL Website

### Deploy and Test on a Local Mac

```shell
brew update
brew install node pnpm
pnpm install
pnpm run dev
```

### Blog Tricks

* href your website url in the author list so readers can contact
* Using reference [^1] and in the blog main text and reference (only work in markdown)
* Using \`\<code\>\` and \```cpp \<lines-of-code\>\``` for code formating
* Using \*\*\<text\>\*\* for bold
